# IPL Data Analysis

A Power BI dashboard project analyzing Indian Premier League (IPL) match data. The project takes raw match records and turns them into visual insights using Power BI.

## Project Contents

- `matches.csv` — Dataset containing IPL match records (teams, venues, results, and other match-level details).
- `IPLDB.pbix` — Power BI report file with interactive dashboards and visualizations built on top of the dataset.

## What This Project Covers

This analysis explores IPL match data to surface insights such as:
- Team performance trends across seasons
- Venue-wise match outcomes
- Win/loss patterns and other match statistics

## How to Use

1. Clone or download this repository.
2. Open `IPLDB.pbix` in [Microsoft Power BI Desktop](https://powerbi.microsoft.com/desktop/) (free download).
3. Explore the interactive dashboard — filter by team, season, or venue to dig into specific insights.
4. If you want to rebuild or modify the analysis, `matches.csv` is the underlying dataset used to power the report.

## Tools Used

- **Power BI** — for building the interactive dashboard
- **CSV** — as the raw data source
